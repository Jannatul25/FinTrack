# 💰 FinTrack — Personal Finance Tracker

> STEM105: Introduction to Programming — Midterm Project

A fully functional personal finance web application built with HTML, CSS, and JavaScript.

## 📋 Features

- ✅ **Add income & expense transactions** with description, amount, category, and date
- ✅ **Monthly navigation** — browse any past month
- ✅ **Summary cards** — Total income, expenses, and net balance at a glance
- ✅ **Category spending bars** — Visual breakdown of where your money goes
- ✅ **6-month bar chart** — Compare income vs. expenses over time
- ✅ **Filter transactions** — View all, income only, or expenses only
- ✅ **Delete transactions** — Remove any entry instantly
- ✅ **Persistent storage** — Data saved in localStorage (no backend needed)
- ✅ **Fully responsive** — Works on desktop and mobile

## ▶️ How to Run

### Option 1 — GitHub Pages (recommended for submission)
1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your app will be live at `https://YOUR-USERNAME.github.io/fintrack`

### Option 2 — Run locally
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR-USERNAME/fintrack.git
   ```
2. Open `index.html` directly in any modern web browser

No build tools, no npm, no dependencies — it just works.

---

## 💡 How to Use

1. **Browse months** using the ← → arrows in the header
2. **Add a transaction** using the sidebar form — choose income or expense, fill in the details, click Add
3. **View your breakdown** in the category bars and 6-month chart
4. **Delete** any transaction with the ✕ button
5. Data is **automatically saved** between sessions
